# Lucent Bidder

[![Build Status](https://travis-ci.org/telefrek/lucent-bidder.svg?branch=alpha)](https://travis-ci.org/telefrek/lucent-bidder)

This is a set of libraries and tools used to setup and run our DSP (Demand Side Platform).  We are currently in the process of refactoring the infrastructure and code and as such, the structure, utilities and build process are likely to experience wild changes.  Thanks for your interest in the project!
